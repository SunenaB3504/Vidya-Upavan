## About Guides & Quizzes

### 📖 Reference Guides
- Each phase includes an **interactive reference guide** (S1-GUIDE, M1-GUIDE, etc.)
- Guides are **NOT videos** — they're built-in reference materials
- They display vocabulary tables, grammar rules, examples, and explanations
- Guides are embedded directly in the app and load with a single click

### 🧩 Interactive Quizzes
- **8 comprehensive quizzes** built into the app for Phases 1-4
- Each quiz has **30 questions** covering that phase's content
- **10 different question types** including:
  - Multiple Choice (MCQ)
  - Fill-in-the-Blank
  - Matching Pairs
  - True/False
  - Short Answer
  - Arrange in Order
  - Letter Recognition
  - Transliteration
  - Translation
  - Pronunciation
- Quizzes feature **smart answer validation** with fuzzy matching (accepts spelling variations)
- Immediate **feedback with explanations** for every question
- **Progress tracking** with localStorage (resume where you left off)
- **Score badges** and pass/fail status (80%+ = pass)
- **Retake unlimited times** to improve your score
