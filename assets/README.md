# Assets Folder

This folder holds static assets for the Vidya Upavan app.

## Required Files (to add before public launch)

| File | Description | How to Create |
| :--- | :--- | :--- |
| `upi-qr-placeholder.png` | **UPI QR Code for Dakshina** | Open GPay/PhonePe → "Receive Money" → Download QR. Replace `YOUR_UPI_ID@bank` in index.html with your actual UPI ID. |
| `icon-192.png` | **PWA App Icon** (192×192px) | Use the Vidya Upavan logo. Square, colourful. Can be created in Canva. |
| `icon-512.png` | **PWA App Icon** (512×512px) | Same as above, larger. |
| `og-banner.png` | **Social Sharing Preview Image** (1200×630px) | Saffron/purple gradient with "Vidya Upavan" + Devanagari text. For WhatsApp link previews. |

## Ko-fi Setup (for International Dakshina)
1. Go to [ko-fi.com](https://ko-fi.com) and create a free account
2. Set display name to "Vidya Upavan"
3. Use the 2-sentence description from the conversation as your bio
4. Copy your Ko-fi page URL and replace `https://ko-fi.com/vidyaupavan` in index.html
